# PROJECT

## Context
This project is **Sharukh Academy**, a course platform focused on Snowflake Job-Oriented Training. 
The exported content currently consists of HTML files initially created in Notion.

## Requirements
- Serve as a static GitHub Pages site representing the complete Sharukh Academy curriculum.
- Maintain a clean, professional aesthetic.
- Allow for structured iterative AI-assisted development across sessions.

## Core Directives
1. All future edits to the curriculum must go through the PAUL loop (PLAN -> APPLY -> UNIFY).
2. Animations and excessive styling have been removed per user preference; maintain the original clean white layout unless explicitly directed otherwise.
